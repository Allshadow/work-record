#### 简述

`CryptoJs`一款前端加密`Js`库

#### 使用

##### `script` 方式引入

1）`cdn`下载地址

```
// cdn 下载地址
https://cdnjs.com/libraries/crypto-js
```

2）已下载的部分`js`文件

```
// 百度网盘
tools/dev/crypto@4.1.11
```

3）引入

感觉`crypto-js.min.js`包不大，简单引入即可

```
<body>
	<script src="./js/crypto-js.min.js"></script>
	<script>
		console.log('cryptoJs', CryptoJS)
	</script>
</body>
```

4）简单的`demo`

```
plugin/demo/202212/01
```
