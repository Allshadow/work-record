#### 安装脚手架

##### 安装

```shell
npm install @vue/cli -g

# 或者
yarn global add @vue/cli
```

##### 升级版本

```shell
npm update -g @vue/cli

# 或者
yarn global upgrade --lasest @vue/cli
```

##### 卸载旧版本 `vue-cli`

```shell
npm uninstall vue-cli

# 或者
yarn global remove vue-cli
```

##### 版本查看

```
vue --version  // 目前 4.x 以上支持创建vue3项目
```

##### 创建项目

```
vue create [name]
```
