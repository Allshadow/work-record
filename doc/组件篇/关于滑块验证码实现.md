#### 简述

整理了一份，关于移动端滑块验证码组件，

如果没有太多的要求，这个是可以拿来即用的！

#### 效果

![image-20211201092327158](关于滑块验证码实现.assets/image-20211201092327158.png)

#### 使用

`<template>`

```
<template>
	<van-button type="danger" @click="toConfirm">滑块验证</van-button>
	<!-- 滑块验证开始 -->
  <slider-verification
    :isShow.sync="sliderShow"
    @done="handleConfirm"
    >
  </slider-verification>
  <!-- 滑块验证结束 -->
</template>
```

`<script>`

```
<script>
import SliderVerification from '@/components/SliderVerification.vue'
export default {
	components: {
    SliderVerification
  },
  data () {
    return {
      sliderShow: false
    }
  },
  methods:{
  	// 滑块验证成功后回调
    handleConfirm (val) {
      this.sliderShow = false
    },

    toConfirm () {
      this.sliderShow = true
    }
  }
}
</script>
```
